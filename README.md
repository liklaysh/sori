## ✨ Что такое SORI

SORI — это self-hosted платформа общения, которая объединяет чаты, голосовые каналы, звонки, медиа и администрирование в единую систему.

Она создана, чтобы заменить разрозненные инструменты одной контролируемой средой:

💬 сообщения в реальном времени  
🔊 голосовые каналы  
📞 личные и групповые звонки  
📎 обмен файлами и медиа  
🛠 встроенная админ-панель  

Никаких внешних сервисов. Никакой зависимости от вендоров.  
Только ваша инфраструктура и полный контроль.

## 🚀 Установка

Для production-style установки нужен чистый сервер:

- Ubuntu 22.04+
- root/sudo доступ
- публичный IPv4
- домен, направленный на сервер
- открытые порты `22/tcp`, `80/tcp`, `443/tcp`, `7881/tcp`, `7882/udp`

Перед запуском установщика создайте DNS `A`-записи:

```text
example.com          -> server public IPv4
api.example.com      -> server public IPv4
livekit.example.com  -> server public IPv4
media.example.com    -> server public IPv4
```

Установка одной командой:

```bash
curl -fsSL https://github.com/liklaysh/sori/raw/main/install.sh | sudo bash
```

Установщик сам проверит сервер, установит Docker / Docker Compose и зависимости, спросит домен, email Let's Encrypt и имя сервера, проверит DNS, настроит Caddy и HTTPS, добавит безопасные UFW-правила, сгенерирует секреты и adminpanel credentials, запустит стек, выполнит миграции/bootstrap и выведет URL, admin URL и данные доступа.

Подробная инструкция: [install.md](install.md).

## 🧠 Архитектура

| Слой | Технологии | Роль |
|------|-----------|------|
| **Frontend** | React 18 + Vite + Zustand | Интерфейс и real-time UI |
| **Backend** | Node.js + Hono | API и бизнес-логика |
| **Database** | PostgreSQL + Drizzle | Хранение данных |
| **Realtime** | Socket.IO + Valkey | События и presence |
| **Voice** | LiveKit | Звонки и голосовые каналы |
| **Storage** | MinIO | Файлы и медиа |
| **Gateway** | Caddy | Роутинг и TLS |

👉 Вся система работает в Docker-контейнерах.
